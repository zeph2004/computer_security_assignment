# Writing Workflow

This document describes the recommended workflow for completing this assignment.

---

# Before Writing

Read the entire question.

Understand exactly what is being asked.

Highlight

- action words
- required tables
- required diagrams
- required examples

---

# While Researching

Keep one browser window for research.

Keep Overleaf open.

Every time a useful source is found

1. Add it to references.bib immediately.

Do NOT wait.

---

# Writing

For every question

Research

↓

Outline

↓

Write

↓

Cite

↓

Compile

↓

Repeat

---

# Every Paragraph

After finishing a paragraph

✓ Compile

✓ Check citations

✓ Read aloud

✓ Check grammar

---

# Every Figure

1. Save inside figures/

2. Give a meaningful filename

Good

pdca-cycle.png

Bad

Screenshot (17).png

---

# Every Table

Always

- caption first

- label second

- reference later

---

# Never Do This

Figure 3 below...

Because later it might become Figure 8.

Instead

```latex
\cref{fig:pdca}
```

---

# References

Never type references manually.

Always

1. Add source to references.bib

2. Cite it

3. Let LaTeX generate the bibliography

---

# End of Each Study Session

Compile.

Fix warnings.

Commit changes (or download a backup).

Write a brief note describing what you completed.

---

# Before Submission

□ Compile with no errors

□ Resolve warnings

□ Verify figures

□ Verify tables

□ Verify bibliography

□ Verify page numbering

□ Proofread

□ Export PDF

□ Submit

---

# Philosophy

Formatting should never interrupt thinking.

If you spend more than five minutes fighting formatting, stop.

There is almost always a cleaner LaTeX solution.
